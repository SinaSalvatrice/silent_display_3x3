the chosen environment is qmk and it's a pro micro controller.
the pins are
rows: 4,5,6
columns: 8,9,10
encoder a and b 16, 14
btn 15

the encoder actions are as followed:
on layer 0 - turning = mousewheel, btn = toggle layer selection
on layer 1 - turning = mousewheel, btn = ctl+s
on layer 2 - turning = volume, btn = mute/unmute
on layer 3 - turning = mousewheel, btn = F23
on layer 4 - turning = RGB brightness, btn = RGB on and off
on layer 5 - turning = select layer, releasing should enter it = toggle layer selection

if there are conflicts occuring, solve them as you find it fitting.

the RGB color depends on the chosen layer, set somehow, I change it later.

the layer is changed with btn+encoder turning.
